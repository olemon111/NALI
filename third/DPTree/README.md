# DPTree PiBench Wrapper

## Installation
1. Set # of merge workers in dptree_wrapper.cpp (optional, default equals to # of worker threads)

```
mkdir build && cd build && cmake ..
make -j 8
```

For long-key support
```
mkdir build && cd build && cmake -DVar_Key=1 ..
make
```
